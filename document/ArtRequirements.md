## 三、占位资源管理

### 3.1 占位资源生成规则
- 使用纯色 PNG 文件作为占位
- 文件名与正式资源保持一致
- 不同类型资源使用不同颜色区分：
  - 角色单位：红色系
  - 敌人单位：蓝色系
  - UI 图标：灰色系
  - 物品/装备：黄色系

### 3.2 占位资源替换流程
1. 美术提供正式资源文件（相同文件名、相同尺寸、相同格式）
2. 替换 `demo/art/` 对应路径下的占位文件
3. 代码无需任何修改，自动生效

## 四、命名规则

### 4.1 角色单位
```
unit_{type}.png
例：unit_fighter.png, unit_ranger.png, unit_mage.png
```

### 4.2 UI 图标
```
icon_{name}.png
例：icon_gold.png, icon_hammer.png, icon_sword.png
```

### 4.3 物品/装备
```
item_{slot}_{rarity}.png
例：item_weapon_normal.png, item_helmet_rare.png
```

### 4.4 特效
```
fx_{name}_{frame:00}.png
例：fx_hit_01.png, fx_hit_02.png
```
